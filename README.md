# 🛡️ Roblox Server Validation Test Suite

> **⚠️ ТОЛЬКО ДЛЯ ТЕСТИРОВАНИЯ В ROBLOX STUDIO!**
> Все скрипты предназначены исключительно для проверки серверной валидации в вашей игре.
> Не используйте их на публичных серверах.

## 📋 Описание

Набор из 23 тестовых скриптов (LocalScript) для проверки серверной защиты в Roblox. Каждый скрипт имитирует конкретный тип нарушения, чтобы убедиться, что ваш сервер корректно обнаруживает и блокирует его.

## 🚀 Как использовать

1. Откройте **Roblox Studio**
2. Запустите тест-сервер (Test → Play)
3. Откройте **Command Bar** (View → Command Bar)
4. Скопируйте содержимое нужного `.lua` файла
5. Вставьте в Command Bar и нажмите Enter
6. Наблюдайте за реакцией сервера в Output

## ⚙️ Управление из CommandBar

Большинство скриптов регистрируют глобальные функции через `_G`, которые можно вызывать из CommandBar:

```lua
-- Примеры использования:
_G.ShowConfig()                    -- текущие настройки
_G.LoadConfig('combat_tests')      -- загрузить пресет
_G.TestSpeed(100, 3)               -- тест скорости 100 на 3 сек
_G.SpamDodge(30, 0.01)             -- спам 30 dodge
_G.TeleportNow('PlayerName')      -- телепорт к игроку
_G.FaceLockTarget('PlayerName')   -- аимлок на игрока
```

## 🔒 Безопасность

- Все скрипты работают **только на клиенте** в Roblox Studio
- Они **не модифицируют** серверный код
- Предназначены для **пенетрейшн-тестирования** вашей серверной валидации
- Каждый скрипт имеет функцию очистки (`_G.Cleanup...()`)
